# One ROM Cases

This directory contains 3D printable cases for One ROMs:

- [case-24-with-header-pins.stl](case-24-with-header-pins.stl) - A case for One ROM 24, with header pins installed.
- [case-28-with-header-pins.stl](case-28-with-header-pins.stl) - A case for One ROM 28, with header pins installed.
- [case-40-with-header-pins.stl](case-40-with-header-pins.stl) - A case for One ROM 40, with header pins installed.
- [cap-with-header-pins.stl](cap-with-header-pins.stl) - A cap for all One ROMs with header pins installed.

All models were designed to be printed using PLA filament.

The cap is designed to be a (tight) friction fit, but it may be wise to tape the cap onto the case, if shipping, to ensure it doesn't work free.
